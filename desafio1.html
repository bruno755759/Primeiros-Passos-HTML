Tags de titulo

As tags Html h1 até h6 são utilizadas para definir diferentes níveis de títulos em uma página. Quanto menor o número, maior a importância do título. O h1 é geralmente reservado para o título principal da página, enquanto os h2 até h6 podem ser utilizados para subtítulos ou hierarquias de informações menos relevantes1. Esses elementos representam seis níveis de título de seção, com o h1 sendo o mais alto e o h6 o mais baixo.

Tags marcação de texto

A tag p é usada para criar parágrafos de texto. Ela envolve o conteúdo que você deseja exibir como um parágrafo.

O elemento blockquote indica que o texto incluído é uma longa citação. Normalmente, o texto dentro de blockquote é processado visualmente com recuo. Você pode usar o atributo cite para fornecer a URL da fonte da citação.

A tag sup é usada para criar texto sobrescrito, que aparece ligeiramente acima da linha normal. É comumente usado para notações matemáticas, como expoentes.

O elemento abbr representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

O elemento del representa uma parte do texto que foi excluída de um documento. É renderizado pelos navegadores com uma linha entre o texto.

Tags de formatação

A tag mark é usada para destacar partes do texto, geralmente aplicando um fundo amarelo para chamar a atenção.

A tag small é usada para indicar que o texto deve ser exibido em um tamanho menor, como notas de rodapé ou informações secundárias.

A tag i é usada para aplicar itálico ao texto. Ela não deve ser usada apenas para estilizar o texto, mas sim para indicar que o conteúdo tem um significado diferente ou ênfase.

A tag u é usada para aplicar sublinhado ao texto. No entanto, seu uso é menos comum atualmente, pois o sublinhado geralmente indica um link.

A tag strong dá ao texto uma forte importância e é tipicamente mostrada em negrito. Use-a para destacar partes do texto que são de grande importância, como alertas ou informações urgentes. Evite usá-la apenas para aplicar estilo negrito; para isso, utilize CSS (font-weight).

A tag sub é usada para criar texto subscrito, que aparece ligeiramente abaixo da linha normal. Pode ser usado para fórmulas químicas, notas de rodapé ou outras situações em que o texto precisa estar abaixo da linha base.

A tag hr cria uma linha horizontal para separar conteúdos em uma página. É frequentemente usada para dividir seções de um artigo ou para criar uma divisão visual.

A tag font costumava ser usada para definir o estilo de fonte em HTML. No entanto, seu uso está obsoleto e não é recomendado. Em vez disso, use CSS para estilizar o texto.

Tags de lista

A tag ol é usada para criar uma lista de itens ordenados. Esses itens são mostrados com uma contagem que os precede, podendo ser numerais, letras, algarismos romanos ou símbolos.

A tag ul cria uma lista de itens sem ordem rígida, ou seja, uma coleção de itens que não segue uma numeração específica e cujas posições são irrelevantes.

A tag li é usada dentro das listas (ol ou ul) para definir cada item da lista.

Tags para buscar links e imagens

O elemento a em HTML, também conhecido como âncora, é usado para criar hiperlinks nas páginas da web. Ele permite que você vincule a outras páginas, arquivos, endereços de e-mail ou até mesmo a âncoras na mesma página. O atributo href dentro da tag a especifica o destino do link.

A tag img é usada em HTML para incorporar imagens em uma página da web. Ela cria um espaço reservado para a imagem referenciada e possui dois atributos obrigatórios: src: Especifica o caminho para a imagem. alt: Define um texto alternativo para a imagem, caso ela não possa ser exibida por algum motivo
